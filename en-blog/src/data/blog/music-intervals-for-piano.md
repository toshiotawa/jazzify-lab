<p>Learning <strong>music intervals for piano</strong> gives you a practical language for describing the distance between notes. That language makes chords, scales, and improvised lines easier to understand because you can analyze their structure instead of memorizing every shape separately.</p>

<p>This beginner guide focuses on one essential skill: counting the numerical size of an interval. Interval qualities such as major, minor, perfect, augmented, and diminished come later.</p>

<h2 id="what-is-an-interval-in-music">What Is an Interval in Music?</h2>

<p>An interval is the distance between two notes. Musicians describe intervals with a number and, when more precision is needed, a quality. For example, a major third, minor third, and diminished fifth all name specific distances.</p>

<p>Why is this useful at the piano? If you can recognize the distance between two notes, you can see how a chord is built, understand the pattern of a scale, transpose a phrase, and compare voicings in different keys. Learning music theory without intervals is a little like traveling without a map.</p>

<h2 id="the-first-rule-count-the-starting-note-as-one">The First Rule: Count the Starting Note as One</h2>

<p>The most common beginner mistake is to count the next note as a first. In interval counting, however, the starting note is already <strong>one</strong>. This is called inclusive counting.</p>

<p>For example, C to D is a second:</p>

<p><strong>C (1), D (2)</strong></p>

<p>The same note to itself is a unison, or first. C to C is a unison. D to D is also a unison.</p>

<aside><strong>Quick note:</strong> Count letter names, not the number of piano keys or semitone steps. That distinction becomes important when you add interval qualities later.</aside>

<h3>Try these basic interval questions</h3>

<p>What is the interval from D to F?</p>
<details>
  <summary>Show the answer</summary>
  <p>D (1), E (2), F (3): a third.</p>
</details>

<p>What is the interval from B up to G?</p>
<details>
  <summary>Show the answer</summary>
  <p>B (1), C (2), D (3), E (4), F (5), G (6): a sixth.</p>
</details>

<p>What is the interval from A up to G?</p>
<details>
  <summary>Show the answer</summary>
  <p>A (1), B (2), C (3), D (4), E (5), F (6), G (7): a seventh.</p>
</details>

<h2 id="how-to-count-intervals-with-sharps-and-flats">How to Count Intervals with Sharps and Flats</h2>

<p>When you are finding only the interval number, ignore sharps and flats for a moment and count the letter names. C to F is a fourth, and C to F-sharp is also some kind of fourth. The accidental changes the interval quality, but it does not change the number.</p>

<p>This method keeps the two parts of an interval name separate:</p>

<ul>
  <li>The letter names determine the <strong>number</strong>.</li>
  <li>The exact number of semitones determines the <strong>quality</strong>.</li>
</ul>

<p>Qualities such as perfect, major, minor, augmented, and diminished will tell us why C-F and C-F-sharp sound different. For now, both span the letters C, D, E, and F, so both are fourths.</p>

<h3>Practice with accidentals</h3>

<p>What is the numerical interval from F-sharp to C-flat?</p>
<details>
  <summary>Show the answer</summary>
  <p>Ignore the accidentals temporarily and count F, G, A, B, C: a fifth.</p>
</details>

<p>What is the numerical interval from E to B-flat?</p>
<details>
  <summary>Show the answer</summary>
  <p>Count E, F, G, A, B: a fifth.</p>
</details>

<h2 id="finding-an-interval-above-a-note">Finding an Interval Above a Note</h2>

<p>The next skill is to name the note a given interval above a starting note. Count the starting note as one and move upward through the musical alphabet.</p>

<p>What note is a fifth above C?</p>
<details>
  <summary>Show the answer</summary>
  <p>C (1), D (2), E (3), F (4), G (5): G.</p>
</details>

<p>What note is a unison above E?</p>
<details>
  <summary>Show the answer</summary>
  <p>E. A unison begins and ends on the same letter name.</p>
</details>

<p>What note is a sixth above F?</p>
<details>
  <summary>Show the answer</summary>
  <p>F (1), G (2), A (3), B (4), C (5), D (6): D.</p>
</details>

<p>Being able to answer these quickly is a foundation for building chords and scales. If a chord formula asks for a third, fifth, or seventh above a root, you first need to know the correct letter name before choosing a sharp or flat.</p>

<h2 id="finding-an-interval-below-a-note">Finding an Interval Below a Note</h2>

<p>Descending intervals use the same inclusive-counting rule, but you say the musical alphabet backward. For example, a fifth below C is F:</p>

<p><strong>C (1), B (2), A (3), G (4), F (5)</strong></p>

<p>If descending letter names feel awkward, practice speaking one octave down from every starting note:</p>

<ul>
  <li>D-C-B-A-G-F-E-D</li>
  <li>E-D-C-B-A-G-F-E</li>
  <li>F-E-D-C-B-A-G-F</li>
  <li>G-F-E-D-C-B-A-G</li>
  <li>A-G-F-E-D-C-B-A</li>
  <li>B-A-G-F-E-D-C-B</li>
</ul>

<p>Do this away from the piano as well as at the keyboard. The goal is to make both ascending and descending letter sequences automatic.</p>

<h3>Descending interval practice</h3>

<p>What note is a sixth below D?</p>
<details>
  <summary>Show the answer</summary>
  <p>D (1), C (2), B (3), A (4), G (5), F (6): F.</p>
</details>

<p>What note is a seventh below E?</p>
<details>
  <summary>Show the answer</summary>
  <p>E (1), D (2), C (3), B (4), A (5), G (6), F (7): F.</p>
</details>

<p>What note is a second below F?</p>
<details>
  <summary>Show the answer</summary>
  <p>F (1), E (2): E.</p>
</details>

<h2 id="a-five-minute-piano-interval-practice-routine">A Five-Minute Piano Interval Practice Routine</h2>

<ol>
  <li>Choose a random starting note.</li>
  <li>Name a second through a seventh above it.</li>
  <li>Play both notes on the keyboard and say the interval number aloud.</li>
  <li>Repeat the exercise below the starting note.</li>
  <li>Add sharps or flats, but continue using the letter names to find the numerical interval.</li>
</ol>

<p>Keep the exercise slow enough to be accurate. Speed comes from repetition. Once the number is reliable, you can add interval quality and connect the same thinking to chord tones, chord extensions, and scale formulas.</p>

<h2 id="how-intervals-lead-to-jazz-piano">How Intervals Lead to Jazz Piano</h2>

<p>Jazz musicians constantly think in relationships: a third and seventh define a chord, a ninth extends it, and a ii-V-I progression moves roots by characteristic intervals. Knowing intervals lets you transfer one musical idea to all twelve keys instead of treating every key as unrelated information.</p>

<p>After you can count confidently on the keyboard, practice the same relationships in musical context. Jazzify can help turn the theory into repetition by letting you work with chords, scales, and progressions as playable material.</p>

<h2 id="summary">Summary</h2>

<ul>
  <li>An interval describes the distance between two notes.</li>
  <li>Count the starting note as one.</li>
  <li>Use letter names to determine the numerical interval.</li>
  <li>Sharps and flats affect interval quality, not the number of letter names spanned.</li>
  <li>Practice finding intervals both above and below every note.</li>
  <li>Interval fluency makes chords, scales, transposition, and jazz improvisation easier to understand.</li>
</ul>
